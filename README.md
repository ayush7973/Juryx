# Juryx
Law Case Management System

Juryx is a full-stack law case management system built using Spring Boot, JWT authentication, Role-based access, REST APIs, and React-based UI.
It includes secure login, user registration, case management, and a clean project structure focused on backend development.


Backend (Spring Boot + JWT)
___________________________

User Registration & Login

JWT Authentication & Authorization

Role-based access control (Admin / Lawyer / Client)

CRUD APIs for Case Management

H2 Database 

Secure Password Hashing


Frontend 
________

Register & Login with JWT token

Protected dashboard

Create and view cases

Simple UI to test all major APIs
